## Deployment

To deploy this project run

```bash
  git clone https://github.com/Navaneethakrishnan2004/Advance-Key-Logger.git
```
Run on terminal or vs code or any Python environment
```bash
  python logger.py
```
```bash
  python Keylogger.py
```
